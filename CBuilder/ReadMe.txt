This is a Cbuilder project, that can compile, and debug the Delphi/Pascal source.

Keep the directories as-is.

XtTools\Cbuilder Contains the CBuilder project.
XtTools\Delphi Contains the Delphi Pascal source

The exe built with Cbuilder also needs the files vl280.bpl and rtl280.pbpl. They can be found in your CBuilder installation.
